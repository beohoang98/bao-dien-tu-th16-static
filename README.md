## Run in localhost

1. **Requirement**

    - **Ruby**
    ```bash
    # Ubuntu
    sudo apt install ruby-dev
    ```

    - **Jekyll and Bundler**
    ```bash
    gem install jekyll bundler
    ```

2. **First Install**
```bash
bundle install
```

3. **Run**
```bash
bundle exec jekyll serve
```
